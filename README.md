This project is a comprehensive data analytics dashboard built in Power BI, focusing on the BMW car sales dataset from 2010 to 2024.
It showcases advanced data modeling, interactive visualizations, and business insights useful for sales performance tracking and strategic decision-making.

Key Features

1. Interactive Power BI Dashboard built using DAX measures & calculated columns

2. Sales performance analysis across multiple years

3. Pricing trends by engine size, mileage, and model

4. Region-wise sales volume breakdowns

5. Fuel type distribution and color popularity analytics

6. Mileage vs. Price correlation represented with advanced charts

7. KPIs for average price, total sales, mileage metrics, and high-value models

8. Clean, user-friendly UI with slicers for model, fuel type, region, year, and color
